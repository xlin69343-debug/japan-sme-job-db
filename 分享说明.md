# 日本全行业中小企业求职数据库网站 - 分享说明

## 最简单的分享方式

把 `japan-sme-job-db-share.zip` 发给别人。

对方解压后有两种打开方式:

### 方式一: 上传到静态网站托管

把 `site/` 文件夹上传到以下任意平台:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- 任意支持静态 HTML 的服务器

上传后入口是 `index.html`。

### 方式二: 本地预览

进入解压后的文件夹, 执行:

```bash
cd site
python3 -m http.server 8899
```

然后打开:

```text
http://localhost:8899
```

## 给编辑者

源项目仍在 `japan-sme-job-db-site/`。

更新数据:

```text
data/companies/*.json
```

重新生成可分享静态网站:

```bash
pnpm run build
```

构建后静态文件会在:

```text
out/
```

## 注意

- 这是 MVP 求职研究版数据库。
- 公司规模、薪资、OpenWork 评分、签证支持、外国人录用情况等字段会变化。
- 正式投递前请以公司官网、採用页、面试确认和在留资格要求为准。
