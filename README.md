# 我的日本求职成长台

这是一个围绕个人成长和日本求职决策搭建的网站。它不是招聘网站，也不是公开社区，而是用来逼自己每天推进一点：C 语言学习、日语准备、公司分层、重点候选和投递记录。

130 家企业只是原始研究库，不代表都适合我、都需要投递。个人使用目标是从 130 家里收敛出 10-20 家主线研究对象，再进一步选出 3-5 家近期重点准备对象。

## 功能

- 今日任务
- 主线候选搜索
- 个人公司分层
- 企业详情页
- 收藏、笔记与投递状态管理
- 每家公司一个 JSON 数据文件
- GitHub Pages 自动部署

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- JSON 数据源

## 本地运行

```bash
pnpm install
pnpm run dev
```

## 数据位置

```text
data/companies/*.json
```

## 发布

推送到 `main` 分支后, GitHub Actions 会自动构建并发布到 GitHub Pages。
