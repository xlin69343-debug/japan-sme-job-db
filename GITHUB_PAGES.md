# GitHub Pages 发布说明

## 推荐方式

创建公开仓库:

```text
japan-sme-job-db
```

推送本项目源码到 `main` 分支后, GitHub Actions 会自动构建并发布 Pages。

最终链接格式:

```text
https://<你的GitHub用户名>.github.io/japan-sme-job-db/
```

## 如果使用个人主页仓库

如果仓库名是:

```text
<你的GitHub用户名>.github.io
```

最终链接格式:

```text
https://<你的GitHub用户名>.github.io/
```

## GitHub 设置

仓库创建后:

1. 打开仓库 Settings
2. 进入 Pages
3. Source 选择 GitHub Actions
4. 等待 Actions 完成
